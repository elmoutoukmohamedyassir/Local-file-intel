🛰️File Intelligence System

it's a high-performance file management and storage intelligence application. Built with **Python** and **Flet**, it provides a high-end dashboard interface to scan, analyze, organize, and optimize local storage through neural-style data processing.


---

## 🚀 Key Features

* **🔍 Neural System Scan:** Rapidly indexes entire directories while filtering out system junk (venv, git, etc.).
* **👯 Deep Duplicate Detection:** Uses **SHA-256 checksum hashing** to find identical files, even if they have different names.
* **📂 Smart Organizer:** Automatically categorizes files into logical groups (Images, Videos, Docs, Setup) based on an intelligent extension map.
* **📊 Storage Intelligence:** Real-time visual cards showing storage distribution and a "Heavy Hitters" list of the largest files.
* **🛑 Instant Interruption:** Integrated "Annuler" logic allows users to safely stop heavy hashing operations mid-process.
* **📜 Activity Console:** A real-time, auto-scrolling log of every system action.

---

## 🛠️ Tech Stack

- **Frontend:** [Flet](https://flet.dev/) (Flutter for Python)
- **Engine:** Python `pathlib` & `hashlib`
- **Concurrency:** Multi-threading for non-blocking UI operations.
